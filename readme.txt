== Description ==

Webgains Ads Widget gives you an ability to fetch ad units from Webgains API, and then display them on your site via widgets.

Full list of features:

* Using your own Webgains account
* Fetch up to 1000 ad units using Webgains API
* Fetch ads from joined/non-joined programs
* Widgets
* Select size of the ads to display
* Select the number of ads to display
* Select to display most recent ads or ads in random order
* Select to display ads only on post/page or both 

Requirements:

* Webgains account
* PHP 5.2.0 or greater

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `webgains-ads-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `'Plugins'` menu in `WordPress`
3. Go to the options panel under the `'Settings'` menu and add your `Webgains account key/campaign id` and start fetching the ads.
